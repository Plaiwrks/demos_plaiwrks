# Plaiwrks — AI Demo Websites

Dit zijn demo-websites voor **Plaiwrks** om aan MKB-klanten te laten zien wat AI kan bouwen via vibe coding.

## Regels

- Elke website is **één zelfstandig HTML-bestand** (embedded CSS + vanilla JS), geen frameworks
- Elke demo krijgt een **eigen submap** onder `/Code/`
- **Stijl:** professioneel en onderscheidend — geen generieke Inter-font of paarse gradiënten
- Altijd **mobiel-first** en **responsive**
- Alle content is **Nederlandstalig**
- Google Fonts via `@import` in de `<style>` tag
- Bewegend beeld via Pexels video's (autoplay, muted, loop)
- Formulieren hebben **Nederlandse foutmeldingen** en client-side validatie
- Scroll-animaties via Intersection Observer (geen externe libraries)
